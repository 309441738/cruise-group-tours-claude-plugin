# cruise-group-tours · Claude Code 插件

为团体旅行提供邮轮产品，满足团队的多样化需求。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-group-tours-claude-plugin
/plugin install cruise-group-tours@cruise-group-tours-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-group-tours**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
